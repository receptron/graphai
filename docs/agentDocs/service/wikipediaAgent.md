# wikipediaAgent

## Description

Retrieves data from Wikipedia

## Schema

#### inputs

```json
{
  "$schema": "http://json-schema.org/draft-04/schema#",
  "description": "",
  "type": "object",
  "properties": {
    "query": {
      "type": "string",
      "minLength": 1
    }
  },
  "required": [
    "query"
  ]
}
```

## Input example of the next node

```json
[
  ":agentId",
  ":agentId.content",
  ":agentId.ns",
  ":agentId.title",
  ":agentId.pageid"
]
```
