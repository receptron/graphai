# lookupDictionaryAgent

## Description

Select elements with params

## Schema

#### inputs

```json
{
  "type": "object",
  "properties": {
    "namedKey": {
      "type": "string",
      "description": "key of params"
    }
  },
  "required": [
    "namedKey"
  ]
}
```

#### output

```json
{
  "anyOf": [
    {
      "type": "string"
    },
    {
      "type": "integer"
    },
    {
      "type": "object"
    },
    {
      "type": "array"
    }
  ]
}
```

## Input example of the next node

```json
[
  ":agentId",
  ":agentId.model",
  ":agentId.temperature"
]
```
```json
[
  ":agentId",
  ":agentId.model",
  ":agentId.temperature"
]
```
