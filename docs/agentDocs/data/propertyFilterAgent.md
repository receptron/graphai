# propertyFilterAgent

## Description

Filter properties based on property name either with 'include', 'exclude', 'alter', 'swap', 'inject', 'inspect'

## Schema

#### inputs

```json
{
  "type": "object"
}
```

#### output

```json
{
  "type": "any",
  "properties": {
    "array": {
      "type": "array",
      "description": "the array to apply filter"
    },
    "item": {
      "type": "object",
      "description": "the object to apply filter"
    }
  }
}
```
