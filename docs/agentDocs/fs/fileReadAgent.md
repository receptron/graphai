# fileReadAgent

## Description

Read data from file system and returns data

## Schema

#### inputs

```json
{
  "type": "object",
  "properties": {
    "array": {
      "type": "array",
      "description": "file names"
    }
  },
  "required": [
    "array"
  ]
}
```

#### output

```json
{
  "type": "object"
}
```
