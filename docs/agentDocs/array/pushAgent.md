# pushAgent

## Description

push Agent

## Schema

#### inputs

```json
{
  "type": "object",
  "properties": {
    "array": {
      "type": "array",
      "description": "the array to push an item to"
    },
    "item": {
      "anyOf": [
        {
          "type": "string"
        },
        {
          "type": "integer"
        },
        {
          "type": "object"
        },
        {
          "type": "array"
        },
        {
          "type": "boolean"
        }
      ],
      "description": "the item to push into the array"
    },
    "items": {
      "type": "array",
      "description": "items to push into the array"
    }
  },
  "required": [
    "array"
  ]
}
```

#### output

```json
{
  "type": "object",
  "properties": {
    "array": {
      "type": "array"
    }
  }
}
```
