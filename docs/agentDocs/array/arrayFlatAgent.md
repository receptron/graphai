## Description

Array Flat Agent

## Schema

#### inputs

```json

{
  "type": "object",
  "properties": {
    "array": {
      "type": "array",
      "description": "The array to be flattened"
    }
  },
  "required": [
    "array"
  ]
}

```

#### output

```json

{
  "type": "object",
  "properties": {
    "array": {
      "type": "array",
      "description": "flattened array"
    }
  }
}

```
