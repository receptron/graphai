## Description

Array Join Agent

## Schema

#### inputs

```json

{
  "type": "object",
  "properties": {
    "array": {
      "type": "array",
      "description": "array join"
    }
  },
  "required": [
    "array"
  ]
}

```

#### output

```json

{
  "type": "object",
  "properties": {
    "text": {
      "type": "string",
      "description": "joined text"
    }
  }
}

```
