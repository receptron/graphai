# images2messageAgent

## Description

Returns the message data for llm include image

## Schema

#### inputs

```json
{
  "type": "object",
  "properties": {
    "array": {
      "type": "array",
      "description": "the array of base64 image data"
    },
    "prompt": {
      "type": "string",
      "description": "prompt message"
    }
  },
  "required": [
    "array"
  ]
}
```

#### output

```json
{
  "type": "object"
}
```

```json
{
  "array": [
    "abcabc",
    "122123"
  ]
}
```

#### params

```json
{"imageType":"png"}
```

#### result

```json
{
  "message": {
    "content": [
      {
        "image_url": {
          "detail": "auto",

        "type": "image_url"
      },
      {
        "image_url": {
          "detail": "auto",
          "url": "data:image/png;base64,122123"

  }
}
```
