# stringTemplateAgent

## Description

Template agent

## Schema

#### inputs

```json
{
  "$schema": "http://json-schema.org/draft-04/schema#",
  "description": "",
  "type": "object",
  "properties": {
    "message1": {
      "type": "string",
      "minLength": 1
    },
    "message2": {
      "type": "string",
      "minLength": 1
    }
  },
  "required": [
    "message1",
    "message2"
  ]
}
```

## Samples

### Sample0

#### inputs

```json
{
  "message1": "hello",
  "message2": "test"
}
```

#### params

```json
{"template":"${message1}: ${message2}"}
```

#### result

```json
"hello: test"
```
### Sample1

#### inputs

```json
{
  "message1": "hello",
  "message2": "test"
}
```

#### params

```json
{"template":["${message1}: ${message2}","${message2}: ${message1}"]}
```

#### result

```json
[
  "hello: test",
  "test: hello"
]
```
