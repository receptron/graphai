# jsonParserAgent

## Description

Template agent

## Schema

#### inputs

```json
{
  "anyOf": [
    {
      "type": "string"
    },
    {
      "type": "integer"
    },
    {
      "type": "object"
    },
    {
      "type": "array"
    }
  ]
}
```

#### output

```json
{
  "type": "string"
}
```
